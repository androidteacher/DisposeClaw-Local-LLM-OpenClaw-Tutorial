⚙️ Local_Chat_UI Setup

Connect your local open-source AI agent to Telegram

Step 1: Create a Telegram Bot

  1. Open the Telegram app and search for @BotFather (the official bot with a blue checkmark).
  2. Send the message /newbot to start the creation process.
  3. Choose a display name for your bot (e.g., My Local AI).
  4. Choose a unique username that ends in "bot" (e.g., MyLocalAIBot).
  5. BotFather will reply with an API Token. It looks like 123456789:ABCdefGHIjklMNOpqrsTUVwxyz. Copy this token.
💡 Tip: Keep this token secret! Do not share it publicly.

Step 2: Enter API Token

We use a secure command-line script to inject the token directly into the running AI container.

cd /home/student/App_Deploy
chmod +x enter_api.sh
./enter_api.sh

Step 3: Pairing Approval

After saving the token, send any message to your new bot on Telegram. Then, you need to approve the pairing request.

Run the following commands manually to view and approve the connection:

# 1. Restart the container to apply the new key
docker restart openclaw

# 2. List pending pairing requests from your device
docker exec -it openclaw openclaw pairing list telegram

# 3. Approve your device connection code (replace <CODE>)
docker exec -it openclaw openclaw pairing approve telegram <CODE>
